find,filter
-------------
filter ==> returns all matching elements
find ==> returns only first matching element

below is the example for above find & filter
let numbers = [10,20,33,40,68];


//filter
let testNo = numbers.filter((n)=>{
    return n > 30;
});

console.log(testNo); ==> 33,40,68


//find

let findGt = numbers.find((n)=>{
return n >30;
});

console.log(findGt) ==> 33
